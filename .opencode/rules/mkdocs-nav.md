# mkdocs.yml Navigation Rules

Applies to: `mkdocs.yml`

## Nav Management
- `nav:` must reflect real files in `docs/`.
- Use short sidebar labels.
- Keep Chapter 2 grouped by act comments.

## 1:1 Mapping
- Every Ch2 topic must have matching Ch3 topic in nav.
- Keep numbering and order coherent.

## Verification Checklist
- Every nav target path exists.
- No orphan docs files outside nav.
- Ch2 and Ch3 mapping remains complete.
