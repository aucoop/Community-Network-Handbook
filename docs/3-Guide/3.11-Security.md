# Security — Hardening Your Network

Practical security measures for community networks.

!!! info "Work in Progress"
    This guide will provide a security checklist and hardening steps for routers, servers, and services.

## What You'll Learn

- OpenWrt firewall configuration and best practices
- Upgrading from WPA2 to WPA3 where supported
- SSH hardening (key-based auth, disable root password login)
- Network segmentation with VLANs (guest, management, IoT)
- Firmware update strategy
- TLS/HTTPS for all web services
- Fail2ban for brute-force protection
