# IP Addressing

A solid IP addressing plan is the invisible foundation of a healthy network. Skip it and you'll pay for it later with address conflicts, unreachable devices, and hours of debugging.

- How to design a subnet plan for multiple buildings and VLANs
- Static vs DHCP — when to use each
- Sample IP addressing plan for a multi-site community network
