# *"Who are all these people on my network?!"*

The network is popular. Too popular. People from the neighborhood are connecting. Students are streaming video all day. Someone is hogging all the bandwidth. You have no idea who's who.

**RADIUS** (Remote Authentication Dial-In User Service) gives you proper user management:

- **Individual accounts** — each user has a username and password
- **Usage tracking** — see who's connected, when, and how much bandwidth they use
- **Access control** — limit speeds, set time quotas, block unauthorized users
- **Multiple SSIDs** — different networks for staff, students, and guests

It's the difference between an open door and a front desk.
