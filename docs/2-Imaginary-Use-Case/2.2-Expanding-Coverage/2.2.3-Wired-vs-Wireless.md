# *"Should I use cables or go wireless?"*

You need to connect multiple routers. There are two main approaches:

**Wired (Switch + Ethernet cables):**

- Reliable, fast, no interference
- Requires running cables through walls, ceilings, or trenches
- Best when you can permanently install cables

**Wireless (Mesh networking):**

- No cables needed — routers talk to each other over WiFi
- Easier to deploy, especially in difficult-to-cable buildings
- Slightly slower and less reliable than wired
- Uses protocols like **802.11s** (mesh) and **802.11r** (fast roaming)

In practice, most community networks use **both**: cables where possible, mesh where cables can't reach.
