# *"What about the building across the street?"*

The main building is fully covered. But the library across the street? The workshop 500 meters away? Regular WiFi won't reach that far.

You need a **point-to-point wireless link** — a dedicated radio connection between two buildings using directional antennas. Think of it as an invisible cable through the air.
