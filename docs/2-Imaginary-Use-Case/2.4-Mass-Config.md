# *"I've been configuring routers all week..."*

You now have twelve routers spread across three buildings. Every time you need to change the WiFi password, update a firewall rule, or tweak a setting, you have to SSH into each one individually. Twelve times. It's maddening.

There has to be a better way — and there is. **OpenWISP** is a network management platform that lets you configure, monitor, and update all your routers from a single dashboard.

Think of it as a remote control for your entire network.
