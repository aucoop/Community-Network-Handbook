# Real Use Cases

Chapter 2 told a fictional story. This chapter tells **real ones**.

These are community networks that have been built, deployed, and are running in the field. Each case study documents:

- **The context** — where, for whom, what problems existed
- **The design** — what technologies were chosen and why
- **The deployment** — what worked, what didn't, lessons learned
- **The current state** — is it still running? How has it evolved?
